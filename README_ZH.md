## 特性/TODO

- Microsoft To-Do macOS客户端
- 原生APP的体验
- 清新易用的界面
  - 界面显示
  - 搜索
  - 清单
  - 任务
  - 设置
- 支持快捷键操作
  - 新建清单 (Cmd + Shift + N)
  - 选择清单 (Cmd + ↑/↓)
  - 新建任务 (Cmd + N)
  - 选择任务 (↑/↓)
  - 搜索/取消搜索 (Cmd + F/ESC)
  - 偏好设置 (Cmd + ,)
- 支持TouchBar操作
  - 新建清单
  - 新建任务
  - 隐藏/显示已完成任务
  - 完成/开始任务
  - 重要/取消重要
  - 删除任务
- 中英文支持
- 暗黑模式（支持macOS 10.14 Mojave的系统设置，其他系统版本可手动设置）✔︎
- 在线升级
- 提示信息
